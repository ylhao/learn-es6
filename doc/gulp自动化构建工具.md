## 为什么我要引入gulp自动构建工具？
在学习过程中，每次修改代码，我都要执行一下“npm run build”，然后再去浏览器刷新一下页面。这些操作看起来还不是很繁琐，但是当工程比较大的时候，这可能就会成为让人头疼的、繁琐的操作。我在想能不能简化这个流程。我修改完代码，自动执行编译过程，并且自动的帮我刷新页面呢？下面尝试用 gulp 来实现这个设想。
